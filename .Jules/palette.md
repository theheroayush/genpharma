## 2024-05-24 - Accessibility for Custom Interactive Upload Zones
**Learning:** Custom interactive `div` elements, such as drag-and-drop upload zones, require native button attributes (`role="button"`, `tabIndex={0}`, and an `onKeyDown` handler) to be fully accessible for keyboard navigation. Additionally, visible focus indicators are essential for keyboard users to identify when an interactive element has focus. Without these attributes, users relying on keyboard navigation cannot interact with the upload zone or trigger file inputs.
**Action:** When implementing custom interactive areas (like drag-and-drop zones), always ensure they are accessible via keyboard by mapping 'Enter' and 'Space' keys to the core interaction (e.g., triggering a file input click). Provide visible `focus-visible` styling for clear keyboard focus indicators.
